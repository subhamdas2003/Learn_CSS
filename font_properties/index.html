Lets Learn about fonts

Fonts are a crucial part of web design, affecting readability and aesthetics. In CSS, you can control font properties to enhance the user experience.

Font Family

The font-family property specifies the typeface to be used. You can list multiple fonts as fallbacks.

p {
    font-family: Arial, sans-serif;
    }

Font Size

The font-size property sets the size of the font. It can be specified in various units like pixels, ems, or percentages.

p {
            font-size: 10px;
        }

Font Weight

The font-weight property defines the thickness of the font. Common values are normal, bold, and numeric values like 100 to 900.

p {
            font-weight: bold;
        }

Font Style

The font-style property allows you to set the style of the font, such as normal, italic, or oblique.

p {
            font-style: italic;
        }

Font Variant

The font-variant property is used to control the display of small-caps or other stylistic variations.

p {
            font-variant: small-caps;
        }
